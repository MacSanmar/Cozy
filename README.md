# Cozy Draw

A fun and relaxing offline drawing webapp. Open `index.html` in any browser to start doodling.

## Features

- **8 creative brushes**: Pencil, Marker, Watercolor, Crayon, Sparkle, Calligraphy, Glow, and Eraser
- **Warm color palette** with custom color picker
- **Adjustable brush size and opacity**
- **Multiple canvas backgrounds** (light, warm, cool, dark)
- **Undo/Redo** with keyboard shortcuts
- **Save drawings** as PNG
- **Ambient floating particles** for a cozy atmosphere
- **Ambient music** generated with Web Audio API (no external files needed)
- **Fully offline** - works without an internet connection (PWA-installable)
- **Touch support** for tablets and phones

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+S` | Save as PNG |
| `E` | Toggle eraser |
| `[` / `]` | Decrease / Increase brush size |

## Usage

Just open `index.html` in a browser. No build step, no dependencies, no server required.
